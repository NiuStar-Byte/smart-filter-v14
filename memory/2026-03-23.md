## Key Technical Details

### DivergenceTracker Class
Tracks dual-write failures for monitoring:
- `record_failure(uuid)`: Log each failure
- `get_gap_size()`: Estimate divergence
- `get_failure_rate()`: Track percentage
- `get_hourly_stats()`: Monitor by hour

### Alert System
- `send_ops_alert(message, severity)`: Notify operators
- Logs to `/tmp/dual_write_alerts.log`
- Severity levels: WARNING, CRITICAL
- Real-time delivery (operators see immediately)

### Default Behavior
- `raise_on_failure=False` (RECOMMENDED)
  - Failures don't halt daemon
  - Signals continue firing
  - Operators alerted
  - Auto-recovery in background
- Alternative: `raise_on_failure=True` (NOT RECOMMENDED)
  - Would halt daemon (not suitable for trading)
