📐 CALCULATION FORMULAS (ALL LOCKED)
═════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

✅ Win Rate = (TP_HIT + TIMEOUT_WIN) / Closed_Trades × 100%
   Where:
   - TP_HIT = count of TP_HIT signals (excluding stale timeouts)
   - TIMEOUT_WIN = timeout signals with P&L > 0
   - Closed_Trades = TP + SL + TW + TL

✅ P&L Calculation (Notional Position = $1,000)
   LONG:  ((exit_price - entry_price) / entry_price) × $1,000
   SHORT: ((entry_price - exit_price) / exit_price) × $1,000

✅ Timeout Classification
   TIMEOUT_WIN: P&L > 0 (exit price favorable to direction)
   TIMEOUT_LOSS: P&L < 0 (exit price unfavorable to direction)

✅ Duration Formatting
   HH:MM:SS format: e.g., "01:34:23" (1 hour 34 min 23 sec)
   Xh Ym format: e.g., "1h 34m" (human readable)

✅ Per-Date Breakdown
   Count: signals on date (fired_time_utc converted to GMT+7)
   Beginning Fired Time: MIN(fired_time) for that date (HH:MM:SS)
   Last Fired Time: MAX(fired_time) for that date (HH:MM:SS)

✅ Aggregation (By any dimension)
   Total: COUNT(signals)
   TP: COUNT(status='TP_HIT' AND NOT stale)
   SL: COUNT(status='SL_HIT' AND NOT stale)
   TIMEOUT: COUNT(status='TIMEOUT')
   Closed: TP + SL + TW + TL
   Open: COUNT(status='OPEN')
   WR: (TP + TW) / Closed × 100%
   P&L: SUM(pnl for all signals in group)
   Avg TP Duration: AVERAGE(duration for status='TP_HIT')
   Avg SL Duration: AVERAGE(duration for status='SL_HIT')
